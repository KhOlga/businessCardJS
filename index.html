Про мене

Отримала повну вищу юридичну освіту: кваліфікація магістр, спеціалізація "правове регулювання економіки"; з 2008 року до 2017 працювала юристом (претензійно-позовна, договірна робота, підготовка правових висновків).

Звільнилась з роботи, оскільки вирішила, що навчанню потрібно приділити максимум вільного часу для того щоб все-таки стати веб-розробником. ↑

Навики

Основи: 1) мови програмування – С, С++, С#, JavaScript, Python; 2) інтерфейси програмування додатків – Windows API, Windows Forms, WPF; 3) мова розмітки – HTML; 4) таблиці стилів – CSS; 5) мова моделювання – UML; 6) мова запитів – SQL; 7) технологія – ADO.NET.

Базові навики роботи з: 1) СКБД (СУБД) – MS SQL Server та програмою SQL Server Managemet Studio від Microsoft; 2) інструментом розробки програмного забезпечення Visual Studio від Microsoft. ↑

Досвід роботи

Отриманий за час роботи юристом досвід дозволяє вільно спілкуватись з представниками клієнта (контрагента), в тому числі оперативно оцінити переваги та недоліки запропонованих умов співробітництва. Маю хороші комунікаційні письмові та усні навички. ↑

Мета

Навчитись створювати веб-сайти та веб-застосунки.

Після навчання отримати роботу початкового рівня в складі професійної команди у великій компанії, що займається веб-розробкою. А в подальшому вирішити, які саме аспекти веб-розробки цікаві для мене. ↑
